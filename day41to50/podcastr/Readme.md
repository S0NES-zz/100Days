## 💻 Instruções para rodar 

### Instalar as dependências React & Next 
```
  npm install
  yarn
```
### Build comando
```
  npm run build
  or
  yarn build 
```
### Rodar o build
```
  npm start
  yarn start  #irá subir a aplicação no localhost:3000
```
### Inicar o servidor
```
  npm server
  yarn server
```
